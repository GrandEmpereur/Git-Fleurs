# Bienvenue sur notre projet Git-Fleur
# About
Suit a notre initiation à notre cours de Git | Github nous avions comme projet de réaliser une extension pour navigateur qui a pour but de re-designer notre portail étudiant.

Si vous est un étudiant du Pôle Léonard de Vinci, vous pouvez suivre les étapes afin de pouvoir bénéficier d'une interface plus sympathique.
#
# Download

<img class="img" src="./img/Git.png"><br>

## 2- Suite à cella, il vous faudra vous rendre dans votre navigateur et ajouter l'extension, puis activer le mode développeur.
<img class="img" src="./img/extantion.png"><br>

## 3- Ensuite, il vous faudra vous rendre dans l'onglet "Charger les extensions non impactées."
<img class="img" src="./img/Extantion_2.png"><br>

## 4- Puis pour finir charger le projet en tant extension.

## Changement
Les principaux changements se verront sur le profil est l'EDT de votre portail étudiant.
